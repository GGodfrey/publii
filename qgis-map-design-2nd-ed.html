QGIS Map Design 2nd Ed

September 28, 2020 by George Godfrey

Inspired by one of the 'recipes' in QGIS Map Design 2nd Ed. and a map I found on Etsy I set out to make something myself. The result is now the banner image for this site.

The base mapping is OS Open Vector roads and buildings layer

There were quite a few components to bring it together. Overall a fun couple of hours nerding.

  • Setting the background colour to black
  • Rotating the map canvas and north arrow in a layout
  • Creating regular rectangles
  • Graduating line thickness and fade
  • Leader lines disappearing to infinity
