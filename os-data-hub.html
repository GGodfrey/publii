OS Data Hub

September 28, 2020 by George Godfrey

OS have just released a new web mapping API. OS Data Hub.

I like the Maps API 'light' style for a subtle background map that doesn't get in the way of your data.

The set up guide is a good general description of adding a WMS layer https://osdatahub.os.uk/docs/wmts/gettingStarted across QGIS, ArcMap, ArcPro, ArcOnline and CADCorp SIS.
